## Navigation Structure

- Dashboard variants dropdown (7 types)
- AI Assistant section (Text/Image/Code/Video generators)
- E-commerce (Products, Billing, Invoices, Transactions)
- Calendar, User Profile, Tasks, Forms, Tables
- Charts, UI Elements, Authentication pages

## Design Patterns

- Card-based layout for each data section
- KPI cards with trend arrows and percentage changes
- Country flag icons for geographic data
- Profile avatars in notification panels
- Dropdown menus throughout
- Modal/card layout for sections
- "See all" links for truncated lists

## Takeaways for CEO Lab

- **KPI card pattern**: 4 top-level metrics with trend indicators — directly applicable to our dashboard (CLMI score, streak, dimensions improved, weekly change)
- **Time period filters**: 12mo/30d/7d/24h toggle for charts — useful for progress tracking views
- **Active users pattern**: Live count + daily/weekly/monthly averages — could adapt for engagement metrics
- **Demographics section with country flags**: Could inspire team/organization comparison views
- **Data table with filtering**: Applicable to dimension scores, assessment history
- **Navigation sidebar with dashboard variants**: Reference for our dashboard nav structure

## THE FRAMEWORK (copywriting.io)

### Structure:
1. **Clarity** - Clear value proposition (what is it?)
2. **Relevance** - Problem agitation (why do I care?)
3. **Value** - Tangible outcomes (what do I get?)
4. **Social Proof** - Testimonials + stats (can I trust it?)
5. **Differentiation** - Why this, not that? (what makes it unique?)
6. **How It Works** - Clear process (is it easy?)
7. **Pricing** - Transparent + value stack (is it worth it?)
8. **FAQs** - Address objections (what about...?)
9. **CTA** - Multiple, strategic (what should I do?)

### Psychological Triggers:
- Loss Aversion (competitors are measuring, you're not)
- Social Proof (500+ CEOs using it)
- Urgency (every month without measurement = falling behind)
- Reciprocity (free trial creates obligation)
- Contrast (vs. courses, vs. guessing)
- Specificity (34% improvement vs. "get better")

### Human Behavior (Lazy, Vain, Busy, Selfish):
- **Lazy:** 8-minute assessments, automated reports
- **Vain:** Track your wins, see your evolution
- **Busy:** Less time than coffee run
- **Selfish:** Your competitive advantage

### The 5 Objections:
1. **No Need** ‚Üí Competitors are measuring
2. **No Desire** ‚Üí Urgency (month without measurement = falling behind)
3. **No Trust** ‚Üí Social proof + free trial
4. **No Money** ‚Üí Value stack ($4K for $15)
5. **No Hurry** ‚Üí Free trial (start today, results in 8 min)
